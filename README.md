# Calculadora Refatorada

Este é um projeto de **calculadora em Java**, feita há algum tempo, que foi **refatorado e reorganizado** para uma estrutura mais limpa e fácil de entender.

O objetivo do projeto é demonstrar o uso de **Java básico**, incluindo **enums** para representar operações matemáticas e separação clara entre lógica de cálculo e interface de entrada do usuário.

---

## Estrutura do Projeto

```
Calculadora/
│
├── src/
│   ├── Main.java        # Classe main para execução do programa
│   ├── Calculadora.java # Lógica e método principal da calculadora 
│   └── Operacoes.java   # Enum com os sinais (+, -, *, /)
└── README.md
```

---

## Funcionalidades

* Operações básicas: **Soma, Subtração, Multiplicação e Divisão**
* Tratamento de **divisão por zero**
* Estrutura modular usando **enum** para operações

---

## Como Rodar

1. Abra o projeto em qualquer IDE Java.
2. Certifique-se de ter um **JDK configurado** (Java 17 ou superior).
3. Compile e execute `Main.java`.
4. Siga as instruções no console para realizar cálculos.

---

## Observações

Este projeto é uma versão **refatorada de um projeto antigo**, com melhor organização de código e uso correto de boas práticas de Java básico.
